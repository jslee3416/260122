# 🌡️ 서울 기온 역대 비교 분석기 (Seoul Temperature Analyzer)

이 프로젝트는 1907년부터 현재까지의 서울 기온 데이터를 활용하여, 특정 날짜의 기온이 과거 동일한 날짜들에 비해 얼마나 더웠는지 또는 추웠는지를 분석하고 시각화해주는 스트림릿(Streamlit) 웹 애플리케이션입니다.

## 🚀 주요 기능
* **데이터 자동 로드**: 118년치(1907년~현재) 서울 기온 데이터를 기본 탑재.
* **사용자 데이터 업로드**: 기상청 자료와 같은 형식의 CSV 파일을 업로드하면 즉시 분석 가능.
* **상대적 기온 비교**: 선택한 날짜의 기온을 역대 평균(평년) 및 순위와 비교.
* **인터랙티브 시각화**: 
    - **기온 분포(Histogram)**: 역대 해당 날짜의 기온 분포 내 현재 위치 표시.
    - **시계열 추이(Line Chart)**: 연도별 해당 날짜의 기온 변화 확인.

## 🛠️ 기술 스택
* **Language**: Python 3.x
* **Framework**: Streamlit
* **Library**: Pandas, Plotly
* **Data Source**: 기상청 기상자료개방포털 (서울 지점)

## 📂 파일 구조
* `app.py`: 스트림릿 웹앱 메인 실행 코드
* `requirements.txt`: 필요한 라이브러리 목록
* `ta_20260122174530.csv`: 서울 기온 데이터셋 (기본 데이터)
